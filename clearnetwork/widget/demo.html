ClearNetwork Widget

Embeddable network status checker for healthcare providers.

Basic Usage

<clear-network-check plan-id="your-plan-uuid" provider-npi="1234567890" api-base="https://api.example.com/v1" ></clear-network-check>

With Alternatives

<clear-network-check plan-id="your-plan-uuid" provider-npi="1234567890" api-base="https://api.example.com/v1" show-alternatives="true" ></clear-network-check>
