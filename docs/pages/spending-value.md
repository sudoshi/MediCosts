# Spending & Value (`/spending`)

**Component:** `client/src/views/SpendingValue.jsx`
**Nav group:** Cost & Financials

## Purpose
Analyzes Medicare spending efficiency — which hospitals and regions deliver the best quality per dollar spent. Correlates CMS per-beneficiary spending scores with quality outcomes.

## Features
- Spending vs quality scatter (efficiency frontier)
- State-level spending benchmarks
- Hospital-level value scores
- Medicare spending per beneficiary (MSPB) analysis

## Data Sources
- `/api/spending` — MSPB scores and quality correlations
