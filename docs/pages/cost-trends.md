# Cost Trends (`/trends`)

**Component:** `client/src/views/CostTrends.jsx`
**Nav group:** Cost & Financials

## Purpose
Time-series analysis of Medicare inpatient costs. Shows how DRG prices, charges, and Medicare payments have changed over the available data years.

## Features
- Multi-year cost trend line charts by DRG
- State-level trend comparisons
- Charge vs payment divergence analysis
- Top DRGs by cost growth

## Data Sources
- `/api/trends` — multi-year DRG cost aggregates
